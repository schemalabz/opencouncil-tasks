# Discord Announcement Template

Short message for the development Discord. Read by community members, contributors, and anyone following the project.

## Structure

```markdown
**OpenCouncil Tasks [version]** is out!

<!-- 1-2 sentences: what's in this release, plain language. Lead with the biggest thing. -->

<!-- 2-3 bullet highlights — only what users/operators will notice. One emoji per bullet max. -->

Full release notes: <github-release-url>
```

## Rules

- **5-10 lines max** — it's a chat message
- **Skip internal/infra changes** unless they affect contributors
- **Plain language**: "logs now survive restarts" not "implemented log persistence with rotation"
- **No @everyone or @here** — let the user decide
- **Always include the release URL**
